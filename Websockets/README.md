# CHAT Socket.IO

Application de chat Socket.IO basee sur le tutoriel officiel.

## Installation

```sh
npm install
```

## Demarrage

```sh
npm start
```

L'application est disponible sur `http://localhost:3000`.

Pour tester le broadcast, ouvrez `http://localhost:3000` dans deux onglets, envoyez un message dans un onglet et verifiez qu'il apparait dans les deux.
